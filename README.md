Shark Attack Data Cleaning & Analysis

Project Overview

This project focuses on cleaning and analyzing a real-world shark attack dataset.

The goal is to transform messy data into a structured dataset and uncover patterns in shark attack risk across activities and locations. These insights are used to support a data-driven safety certification concept for coastal businesses.

Analysis Approach

The analysis focuses on identifying key risk patterns based on:
	•	Activities (e.g. surfing, swimming)
	•	Geographic distribution (countries)
	•	Injury severity (fatal vs non-fatal outcomes)

⸻

Key Insights
	•	Most shark incidents occur during surfing and swimming
	•	Incidents are concentrated in a small number of countries (e.g. USA and Australia)
	•	Risk severity varies depending on the activity

Conclusion

The analysis shows that shark attack risk is not random but follows clear patterns based on activity and location.

These insights can be used to support data-driven decision-making and enable safety-focused applications for businesses such as surf schools and coastal hotels.
